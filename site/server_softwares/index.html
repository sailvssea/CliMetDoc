    Virtual Network Computing (VNC)

    Configuration at Server Point

    1. Log in the server using ssh command on your local computer:
      ssh [username]@[xxx.xxx.xxx.xxx server ip address]

    2. Set the password for the vnc connection on your local computer:
      vncpasswd

    3. Start the vnc server:
      vncserver -localhost no

    4. Check the vnc server status and the port # (DISPLAY #):
      vncserver -list

    Configuration at Client Point

    You can use the vnc viewer client to connect the vnc server created as above.
    vnc viewer clients

    • TigerVNC viewer
    • RealVNC viewer

    Stop the vnc Server

    If you want to stop vnc server, you just kill the server via terminal command on your local computer.
    vncserver -kill:*
    '*' is for all vnc servers; you can use number to kill a specific port.
